# Push Service

## Install

Install binary with go get:

```
go get gitlab.fixb.com/mir/push
```

Then run:

```
./push --config=/path/to/config.conf
```


## Develop

1. Clone repository

```
git clone git@gitlab.fixb.com:mir/push.git
```

2. Run server

```
go run main.go --config=default.conf
```
